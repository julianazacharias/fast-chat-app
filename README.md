## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js installed

### Running the Development Server

To start the development server, run:

```bash
npm install

```

```bash
npm run dev

```

### Running tests

To run the tests:

```bash
npm run test

```

or

```bash
npm run test:ui

```

### Building for Production

To build the project for production, run:

```bash
npm run build

```
